Simple CSS Masonry Grid

~/writing/2025/masonry-grid Jul 2025
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Simple, CSS only Masonry grid. Just replace the amount of columns and you’re done. Use however you want. One issue: The items go down vertically.

/* Container */
div {
	columns: 2;
}
/* Items */
div > * {
	break-inside: avoid;
}
