# Part 1, What whats pay per crawl?

On July 1<sup>st</sup>, Cloudflare proposed a new monetisation scheme for AI crawling websites. ([here](https://blog.cloudflare.com/introducing-pay-per-crawl/)) It proposes the reintroduction of the HTTP status code [402: Payment Required](https://developer.mozilla.org/en-US/docs/Web/HTTP/Reference/Status/402), which never ended up getting adopted. Pay per crawl suggests that instead of monetising web articles through invasive tracking and annoying ads, the publisher / author would charge a small amount of money, say up to a cent, per article.
